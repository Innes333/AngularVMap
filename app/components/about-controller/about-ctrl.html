About VMaps GeoData Visualizer

GeoData Visualizer provides a secure visualization of project data. All data is securely protected. Access can be obtained only by an authorized user.
This version of the GVD supports the following functions:
  1. Zoom function
  2. Display of geo data (points, lines, polygons)
  3. Switching between layers
  4. Geo Data is cached and give an ability to work offline
Will be implemented soon:
  1. Search by attributes in all layers
  2. GVD is going to be presented like apk, that will increase UX
It is possible to work in offline mode. To do this, you need to open the portal when connecting to the Internet and continue to work without connecting to the Internet. Enjoy simple and convenient visualization of your data.
